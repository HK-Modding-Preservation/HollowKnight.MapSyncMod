- [Benchwarp](https://github.com/homothetyhk/HollowKnight.BenchwarpMod). Bench warp
- [RandoMapMod](https://github.com/syyePhenomenol/RandoMapMod).. Bench warp. MapSync (Transition 2 Visited Only)
